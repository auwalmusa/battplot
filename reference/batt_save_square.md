# Save a square plot for LinkedIn

Save a square plot for LinkedIn

## Usage

``` r
batt_save_square(plot, filename, size = 8, dpi = 300)
```

## Arguments

- plot:

  A ggplot object.

- filename:

  Output file path, should end with .png

- size:

  Size in inches, applied to width and height.

- dpi:

  Resolution in dots per inch.

## Value

The filename, invisibly.
