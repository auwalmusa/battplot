# Standard Battlytics labels

Standard Battlytics labels

## Usage

``` r
batt_labs(title, subtitle = NULL, caption = NULL, x = NULL, y = NULL)
```

## Arguments

- title:

  Plot title.

- subtitle:

  Plot subtitle.

- caption:

  Plot caption, usually includes source.

- x:

  X-axis title.

- y:

  Y-axis title.

## Value

A ggplot2 labels object.
