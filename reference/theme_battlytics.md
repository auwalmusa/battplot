# Battlytics ggplot2 theme

A clean, consulting-style theme for charts used in Battlytics briefs.

## Usage

``` r
theme_battlytics(base_size = 12, base_family = "sans")
```

## Arguments

- base_size:

  Base font size.

- base_family:

  Base font family, default "sans".

## Value

A ggplot2 theme object.
